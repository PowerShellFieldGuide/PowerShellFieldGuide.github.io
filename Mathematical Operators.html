Mathematical Operators

What is an Operator?

Now that you understand datatypes, we can start to use these fundamental building blocks to start doing things. First, a simple math review: 5 + 5 is called a math expression. Math expressions consist of two numbers and an operator (+). You can use PowerShell expressions to perform calculations in the shell or as part of a script.

Operators

Symbol Name Description
+ Addition Add two numbers; Concatenate strings
- Subtraction Subtract two numbers
- Minus Make a number negative
* Multiplication Multiply two numbers; Copies strings
/ Division Divide two numbers
% Modulus Divides numbers and returns remainder

Demonstrations

Addition

Here we have an example of how the addition operator works in multiple scenarios. First, we add the numbers 3 and 4, and like you would expect the output is 7. One detail that may not be so obvious is that we are adding to integers here and the operator returns an integer with the value of 7. Next, we use the addition operator to join two strings together. This is called concatenating two strings (fun fact: the Unix command cat is short for concatenate!).

Next up, we use the addition operator with a string and an integer. In this case, PowerShell treats the integer as a string and concatenates it with the string. This is an example of operator overloading which PowerShell supports. Operator overloading means the operators in a language can be used for different purposes depending on the arguments they are provided. In this case, the addition operator can add numbers and also join strings. While this is something many programming languages support, not all will support operator overloading. The last three examples demonstrate this concept using various numeric types and a string.

Subtraction

Next we have subtraction. This operator subtracts two numeric data types. In case you were wondering what happens if you try to subtract a string from a number, PowerShell returns an error.

Minus

In addition to subtraction, we also represent minus with this operator.

Multiplication

Up next, we have multiplication. This operator, like addition, behaves differently depending on whether it is given two numeric types or a numeric type and a string. In the first case (two numeric types as arguments), the operator multiplies the numbers and returns the values. When provided a string and a numeric type, it will copy the string. Please note, this will only work if you provide the string as the first parameter. If you provide the integer as the first argument and a string for the second argument, it will error out.

Division

The division operator is used to divide two numeric types.

Modulus

There is one last operator which you may or may not have heard of before depending on what background you have. This is the modulus (sometimes called "modi") operator. This operator is indispensible and is used to return the remainder of a division operation. To walk you through this a bit, take 31 and divide it by 16. You have 15/16 left over. In this case, modulus will return the value 15.

Operator Precedence
Precedence Operator Description
1 () Parentheses
2 - Minus
3 *, /, % Multiplication and division (and modulus!)
4 +, - Addition and subtraction
