Comparison Operators

What is a comparison operator?

In addition to mathematical operators, PowerShell also has comparison operators. Comparison operators are used to compare two different values or objects to check for things like equality and patterns. PowerShell provides 4 types of comparison operators, these are equality, matching, containment, replacement, and type. For now, we are just going to cover the ones concerning equality.

Equality Operators

Name Syntax
-GT Greater than
-LT Less than
-EQ Equal to
-GE Greater than or equal to
-LE Less than or equal to
-NE Not equal to

This is one of the things in PowerShell if you're coming from a programming background that might be a little bit hard to adjust to. Normally, these operators are represented with their mathematical symbols, but in PowerShell they look more like switches for command line utilities. They do in fact work this way when filtering objects in cmdlets like Get-ADUser, in case you were wondering why Microsoft would think this was a good idea. This is the first example we are going to encounter where Microsoft deviates from a convention most programming languages use in order to be easier to use as a shell language.
