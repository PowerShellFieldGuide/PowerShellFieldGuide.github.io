What is an arraylist?

There is a hybrid data structure called an arraylist that you may already know if you have experience with Java. It is part array and part list and is a common data structure to encounter in other languages. You may see arraylists in other people's examples and PowerShell may even use them at times behind the scenes.

Don't use arraylists

Arraylists are deprecated in the .Net framework at the time of writing this article. Microsoft's documentation notes this and links to the documentation on lists. This is not because of security concerns, as far as I'm aware, but because the list data type has better performance than the arraylist. I am going to show you some examples with the arraylist data structure for legacy reasons, but you should not use this in your code.

Example 1

This example is similar to the examples in our last lesson. We create the arraylist with similar, and also simpler, syntax to the way we create lists. We add elements with the .add() method and remove elements with .remove() method and access element values as we did with our other collections. One difference is that the .add() method returns the element index and the .remove() method does not return true or false. Also, do note that arraylists can store any data type.
